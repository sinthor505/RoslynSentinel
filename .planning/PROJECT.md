# Project: Roslyn Sentinel

## Core Value
Create a high-performance, stateful MCP (Model Context Protocol) server that provides AI agents with 'Compiler-Grade' intelligence and atomic refactoring capabilities for .NET solutions.

## Vision
To bridge the gap between AI code generation and reality by providing a persistent, semantically-aware host that guarantees code validity through the Roslyn compiler.

## Constraints
- Language: .NET 10 / C#
- Framework: Microsoft.CodeAnalysis (Roslyn)
- Protocol: Model Context Protocol (MCP)
- Runtime: Persistent daemon process via stdio/HTTP transport
