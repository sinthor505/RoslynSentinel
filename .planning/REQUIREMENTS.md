# Requirements

## V1 Scope

### CORE - Persistent Host
- **REQ-CORE-01**: Robust solution management (load/unload .sln)
- **REQ-CORE-02**: Partial reloading and FileSystemWatcher integration
- **REQ-CORE-03**: Health monitoring (memory usage, Roslyn health)

### VAL - Semantic Validation Loop
- **REQ-VAL-01**: Efficient in-memory branching for speculative compilation
- **REQ-VAL-02**: Rapid diagnostic reporting (return errors/warnings)
- **REQ-VAL-03**: Performance optimization (semantic model caching)

### ANA - Blast Radius Reporter
- **REQ-ANA-01**: Deep semantic tracing (Find All References across projects)
- **REQ-ANA-02**: Impact analysis (what breaks if signature changes)

### REF - AI-Native Refactoring Macros
- **REQ-REF-01**: MoveTypeToFile atomic refactor
- **REQ-REF-02**: SyncInterface atomic refactor
- **REQ-REF-03**: AutoImport missing namespaces

### DEV - Integration & Dev Experience
- **REQ-DEV-01**: Installation scripts (PowerShell/Bash)
- **REQ-DEV-02**: MCP configuration/registration automation
- **REQ-DEV-03**: Server and tool usage documentation
