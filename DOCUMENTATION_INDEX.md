## 🎯 Quick Reference

### For Users: "Which tool should I use?"
1. Start with **README.md** → Browse tool categories
2. When **TOOL_DOCUMENTATION.md** is available → Read pre/post examples to see exact transformations

### For Users: "Is this tool complete?"
1. Check **UNFINISHED_FEATURES.md** → Search tool name
2. If listed under "Known Limitations" → Read the workaround
3. If listed under "Deferred Bugs" → Has regression test marked `[Ignore]`, awaiting fix

### For Developers: "What still needs to be done?"
1. Read **UNFINISHED.md** → Lists all planned enhancements by difficulty
2. Check **UNFINISHED_FEATURES.md** → Lists deferred bugs and limitations
